# Team Frontend Standards

These standards apply to all frontend code in this project and take precedence 
over general conventions where they conflict.

## Colors
- Primary font color must be #000042 — do not use black (#000000), near-black, 
  or any other dark color for body text. For primary button, use color: #000042 and hover #00E2C1 
- Do not use pure black (#000000) for body text
- Heading-1: font-size: 24px, font-weight: 500, line-height: normal, text-transform: Sentence case
- Heading-2: font-size: 20px, font-weight: 450, line-height: normal, text-transform: Sentence case
- CTA Link Text: font-size: 16px, font-weight: 500, line-height: normal, text-transform: uppercase, line-spacing: 1.6px

## Naming Conventions
- React components must use PascalCase
- Variables and functions must use camelCase
- CSS classes must use kebab-case
